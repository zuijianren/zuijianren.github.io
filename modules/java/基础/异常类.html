异常 ​

继承关系 ​

Error 类 ​

  • 示例: OutOfMemoryError

一般是指与虚拟机相关的问题，如 系统崩溃，虚拟机错误，内存空间不足，方法调用栈移除 等
对于这类错误导致的应用程序中断，仅靠程序本身无法恢复和预防
遇到这种错误，一般情况下，建议直接让程序中止。

Exception 类 ​

  • 示例: NullPointerException

表示的是程序可以处理的异常，可以捕获且能恢复
遇到这类异常，应该尽可能处理异常，使程序恢复运行，而不应该随意终止程序运行

  • Exception 分为:
    • checkedException （受检查异常） => 此处代表的是类型 而不是具体类 Java中实际是不存在 CheckedException 类的, 不能根据 RuntimeException 想当然的认为存在这样的类
    • runtimeException （运行时异常） => 同上, 此处代表的是类型

父子类的异常处理 ​

  • 如果父类抛出了多个异常，子类重写父类方法的时候:
    声明抛出和父类相同的异常或者是父类异常的子类或者不抛出异常, 进行捕获处理
  • 父类方法中没有抛出异常，子类重写父类该方法的时候:
    不能抛出异常，此时子类产生异常，只能捕获处理，不能声明抛出

自定义异常 ​

自定义异常必须继承 Exception 或者 RuntimeException

  1. 继承Exception：那么自定义的异常类就是一个编译器异常，如果方法内部抛出了该异常，必须处理这个异常
  2. 继承RuntimeException:那么自定义的异常类就是一个运行期异常，无需处理，交给虚拟机处理（中断处理）

示例代码 ​

java
public class MyException extends RuntimeException {

    private String message;

    public MyException(String message) {
        super(message);
        this.message = message;
    }
}

try/catch/finally ​

  • finally 语句百分百会执行
  • 如果 finally 前的语句中包含了 return 语句，则会优先执行 finally 中的代码后再返回
  • 如果 finally 中包含 return 语句，那么将不会再去执行前面代码中的 return 语句，而是直接返回
java
public class ExceptionTest {

    public static void main(String[] args) {
        ExceptionTest exceptionTest = new ExceptionTest();
        exceptionTest.test();
    }

    public void test() {
        System.out.println(method());
        System.out.println();
        System.out.println(method1());
        System.out.println();
        System.out.println(method2());
        System.out.println();
        System.out.println(method3());
    }

    /**
     * try语句块的返回语句执行前是否执行finally的语句
     */
    public String method() {
        System.out.println("=====method=====");
        try {
            System.out.println("try执行");
            return "try返回";
        } catch (Exception e) {
            System.out.println("catch执行");
            return "catch返回";
        } finally {
            System.out.println("finally执行");
        }
    }

    /**
     * catch语句块的返回语句执行前是否执行finally的语句
     */
    public String method1() {
        System.out.println("=====method1=====");
        try {
            System.out.println("try执行");
            return "try返回";
        } catch (Exception e) {
            System.out.println("catch执行");
            return "catch返回";
        } finally {
            System.out.println("finally执行");
        }
    }

    /**
     * try和finally的返回语句会选择哪个
     */
    public String method2() {
        System.out.println("=====method2=====");
        try {
            System.out.println("try执行");
            return "try返回";
        } catch (Exception e) {
            System.out.println("catch执行");
            return "catch返回";
        } finally {
            System.out.println("finally执行");
            return "finally返回";
        }
    }

    /**
     * catch和finally的返回语句会选择哪个
     */
    public String method3() {
        System.out.println("=====method3=====");
        try {
            System.out.println("try执行");
            throw new IOException();
        } catch (Exception e) {
            System.out.println("catch执行");
            return "catch返回";
        } finally {
            System.out.println("finally执行");
            return "finally返回";
        }
    }
}
